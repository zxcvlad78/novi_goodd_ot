
На командной строке
Чтобы создать release keystore с помощью инструмента keytool, нужно запустить команду:

keytool -genkey -v -keystore my-release-key.keystore -alias my-key-alias -keyalg RSA -keysize 2048 -validity 10000

В команде нужно указать свои значения: 

my-release-key.keystore — название хранилища ключей. Если нужно создать хранилище в определённом каталоге, а не в текущем, указать полный путь.
my-key-alias — псевдоним ключа подписи.
Ввести пароль для хранилища ключей, параметры владельца (имя, департамент, организация, город, страна — часть можно пропустить) и пароль для ключа подписи.
